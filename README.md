# Máquina de Tradução Neural para Linguagem Informal: um estudo do Inglês ao Português

Este repositório contem código, modelos e conjuntos de dados reunidos para meu último ano do curso de bachalerado de Ciências da Computação na Universidade Estadual do Centro-Oeste (UNICENTRO)

O objetivo deste projeto foi o desenvolvimento e avaliação de máquinas de tradução inglês-português com foco na tradução de textos informais.

Os principais arquivos do projeto são listados a seguir:
```
├── algoritmo_genetico/               <- Algoritmo genético proposto para seleção de frases
├── conjuntos_de_dados/               <- Conjunto de dados reunidos para este trabalho, utilizando frases obtidas pelo site [Tatoeba](https://tatoeba.org/), [Paracrawl](https://paracrawl.eu/), [FAME](https://github.com/laniqo-public/fame-mt) e dados criados manualmente.
├── modelos/               <- Modelos treinados neste trabalho,
└── README.md
```
